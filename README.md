📊 Análise de Vendas de Supermercado

Projeto de análise exploratória de dados utilizando Python, Pandas e Matplotlib.

Objetivo

Analisar os dados de vendas de um supermercado para identificar padrões e gerar insights.

Tecnologias

- Python
- Pandas
- Matplotlib

Análises realizadas

- Faturamento total
- Quantidade de vendas
- Linha de produto com maior faturamento
- Cidade com maior faturamento
- Forma de pagamento mais utilizada
- Média de avaliação por linha de produto

Visualizações

O projeto gera gráficos de:

- Faturamento por linha de produto
- Faturamento por cidade
- Distribuição das formas de pagamento

Como executar

```bash
pip install pandas matplotlib
python analysis.py
